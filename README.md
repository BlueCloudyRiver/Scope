## PebbleTasks - TasksApp

A simple CLI Task Manager app built with python

## Features
- Add Tasks
- Show Tasks
- Save Tasks using JSON
- Mark Tasks as complete
- Delete Tasks

## Technologies Used
- Python
- JSON
- File handling

## How to Run

Clone the repository:

```bash
git clone https://github.com/BlueCloudyRiver/Scope
```

Run the program:

```bash
python main.py
```
